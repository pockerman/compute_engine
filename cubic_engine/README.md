# CubicEngine

## Dependencies

- C++17 compatible compiler
- CMake
- Blaze Linear Algebra library
- OpenBLAS
- GTest
- Boost


You can install OpenBLAS on Ubuntu by

```
sudo apt-get install libopenblas-dev
``` 

## Implementations

### Integrators

- Forward Euler
- RungeKutta 4

### Filtering and Estimation

- Kalman Filter
- Extended Kalman Filter

### Control

- PID Control
